## What is XML?
-Exchanging data between applicaitons in a structured formmated method with common rules and syntax
-Example:
~~~
<firendsList>
  <friend>
    <name>Alex</name>
  </friend>
  <friend>
    <name>Mary</name>
  </friend>
  <friend>
    <name>Rachel</name>
  </friend>
  <bestfriend>
    <name>Ana</name>
  </bestfriend>
</friendslist>
~~~
- XML stands for eXtensible Markup Language which uses tags or <> marks
